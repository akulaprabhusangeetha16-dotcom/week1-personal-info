# Personal Information Manager

## Project Description
This is my first Python project.  
The program stores and displays personal information using Python variables, user input, and formatted output.

---

## What I Learned
- Variables: Storing different types of data (strings, integers)
- Input/Output: Getting input from the user and displaying results
- String Formatting: Using f-strings for clean and readable output
- Error Handling: Basic input validation using while loops
- Simple calculations using Python

---

## Features
- Stores static information (name, age, city, hobby)
- Takes user input (favorite food and favorite color)
- Validates user input (no empty values)
- Displays formatted personal information
- Calculates age in months
- User-friendly welcome and goodbye messages
